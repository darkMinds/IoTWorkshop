Cannabisitter

What is Cannabisitter?

Cannabisitter is an automatic home irrigition solution for plants. It provide you the opportunity to grow up plants without worried about the plant state, its assure that your plant will receive water by some parameters. Using Raspberry Pi 2 and some hardware components we are able to monitor your plant and watering the plant when it needed. You can manged your plants using a mobile platform UI that provide you add a new planet, statistics about your plants state and more.

Server

The server was built using Microsoft Visual Studio and was written in C#.
Our server components:

1. Microsoft Azure Portal
Azure component managed by "Function App", the server uses the Azure IOT hub system to communicate with the user interface app and with the Raspberry Pi. The server contains azure SQL Database that provide us managing data for our product and store the events from the iot hub for statistics.
2. Raspberry Pi UWP program
This program run on Raspberry Pi and monitoring the plant's state using hardware sensors. The data from this program send to azure cloud and if it needed, the azure send "watering command" to Raspberry Pi and the actuator provide water for our plant.

User Interface

The UI component a mobile application that provide the user monitored and managed his plants.
The user can login to his erea, add a new plant for his plants list and control the plants monitoring.
In addition, we provided to the user some statistics about it's plants state: Watering, temperature and humidity.

Sensors

Raspberry Pi 2 with Wifi Dongle, installed with Windows 10

The Raspberry Pi 2 is a credit card sized mini personal computer that lets software developers dive right in and start coding - no additional components needed. Raspberry has its roots in education but should be considered for any purposes where you would otherwise use a PC. In our product, the RP2 acts at the brain: it is responsible for giving out commands to the various components and communicating with the cloud and through it, with the user.

Temperature Sensor

TMP102

The TMP102 Temperature sensor measure the temperature and send the measurment using I2C Protocol. the sensor connect to I2C enters of the Raspberry Pi, we measure the temperature of the plant and send the data to azure with the IOT-HUB.

Soil Moisture Sensor

SEN13322

The soil moisture sensor is an analog device, it measure the soil moisture and we convert his measurment using Analog to Digital Converter: ADC – MCP3008, the data sends to the Azure using IOT-HUB.

Tap Irrigation System

Tap Irrigation Actuator

A solenoid valve is an electromechanically operated valve. The valve is controlled by an electric current through a solenoid. We used a 12V power supply controlled by a BJT.
The actuator releases water when azure function send to RP2 a command for watering following plant state that measure from RP2(temperature and soil moisture measurments).
